RegPulse Lint 规则清单（建议版）


  一、命名规范（Naming）

  ┌─────┬────────────────────────────────┬─────────┬─────────────────────────────┐
  │  #  │              规则              │  级别   │            示例             │
  ├─────┼────────────────────────────────┼─────────┼─────────────────────────────┤
  │ N1  │ 寄存器名全大写                 │ warning │ ctrl → 建议 CTRL            │
  ├─────┼────────────────────────────────┼─────────┼─────────────────────────────┤
  │ N2  │ Field 名全大写                 │ warning │ enable → 建议 ENABLE        │
  ├─────┼────────────────────────────────┼─────────┼─────────────────────────────┤
  │ N3  │ 名字不得使用 Verilog/SV 关键字 │ error   │ input, wire, reg, module 等 │
  ├─────┼────────────────────────────────┼─────────┼─────────────────────────────┤
  │ N4  │ 名字不得以数字开头             │ error   │ 1ST_REG                     │
  ├─────┼────────────────────────────────┼─────────┼─────────────────────────────┤
  │ N5  │ 名字只允许 [A-Z0-9_]           │ warning │ 禁止 -, 空格, 中文          │
  ├─────┼────────────────────────────────┼─────────┼─────────────────────────────┤
  │ N6  │ 名字长度不超过 32 字符         │ warning │ 过长影响生成代码可读性      │
  ├─────┼────────────────────────────────┼─────────┼─────────────────────────────┤
  │ N7  │ 同一 bank 内名字避免过度相似   │ info    │ CTRL 和 CTRL1 并存          │
  └─────┴────────────────────────────────┴─────────┴─────────────────────────────┘

  二、地址与布局（Address & Layout）

  ┌─────┬──────────────────────────────┬─────────┬──────────────────────────┐
  │  #  │             规则             │  级别   │           示例           │
  ├─────┼──────────────────────────────┼─────────┼──────────────────────────┤
  │ A1  │ 地址按 data_width/8 对齐     │ error   │ 32 位下 offset=0x003     │
  ├─────┼──────────────────────────────┼─────────┼──────────────────────────┤
  │ A2  │ 地址空洞检测                 │ warning │ 0x00, 0x04 之后跳到 0x20 │
  ├─────┼──────────────────────────────┼─────────┼──────────────────────────┤
  │ A3  │ 地址空间利用率 < 50%         │ info    │ 提示可能浪费             │
  ├─────┼──────────────────────────────┼─────────┼──────────────────────────┤
  │ A4  │ 首地址建议从 0x0 开始        │ info    │ 非强制                   │
  ├─────┼──────────────────────────────┼─────────┼──────────────────────────┤
  │ A5  │ 地址不超过 block 声明的 size │ error   │ 溢出                     │
  └─────┴──────────────────────────────┴─────────┴──────────────────────────┘

  三、字段与位宽（Field & Bits）

  ┌─────┬──────────────────────────────────────────┬─────────┬─────────────────────────┐
  │  #  │                   规则                   │  级别   │          示例           │
  ├─────┼──────────────────────────────────────────┼─────────┼─────────────────────────┤
  │ F1  │ Reset 值不得超过字段位宽                 │ error   │ 3-bit 字段 reset=0xF    │
  ├─────┼──────────────────────────────────────────┼─────────┼─────────────────────────┤
  │ F2  │ 字段位宽不得超过 data_width              │ error   │ 32 位宽下 field 位宽 40 │
  ├─────┼──────────────────────────────────────────┼─────────┼─────────────────────────┤
  │ F3  │ 保留位（reserved）应命名为 RSVD/RESERVED │ warning │ 未命名的空洞            │
  ├─────┼──────────────────────────────────────────┼─────────┼─────────────────────────┤
  │ F4  │ 单寄存器内字段数过多（>16）              │ info    │ 可读性提示              │
  ├─────┼──────────────────────────────────────────┼─────────┼─────────────────────────┤
  │ F5  │ bit 0 不建议留空                         │ info    │ 习惯性建议              │
  └─────┴──────────────────────────────────────────┴─────────┴─────────────────────────┘

  四、Access Type 语义（Access Semantics）

  ┌─────┬──────────────────────────────────────────┬─────────┬──────────────────────────────────┐
  │  #  │                   规则                   │  级别   │               示例               │
  ├─────┼──────────────────────────────────────────┼─────────┼──────────────────────────────────┤
  │ X1  │ RO 字段 reset 值应为 0（由 HW 驱动）     │ warning │ RO 字段 reset=0xA                │
  ├─────┼──────────────────────────────────────────┼─────────┼──────────────────────────────────┤
  │ X2  │ RO 字段必须有 HW input 端口              │ error   │ RO 但没标 input                  │
  ├─────┼──────────────────────────────────────────┼─────────┼──────────────────────────────────┤
  │ X3  │ WO 字段不应配 HW output                  │ warning │ 软件写入读不出来，配 output 可疑 │
  ├─────┼──────────────────────────────────────────┼─────────┼──────────────────────────────────┤
  │ X4  │ W1C/W1S/W0C 应当有 HW input（status 源） │ warning │ 否则软件无事可清                 │
  ├─────┼──────────────────────────────────────────┼─────────┼──────────────────────────────────┤
  │ X5  │ 单寄存器内混用不兼容 access 类型         │ warning │ RW + W1C 混用语义歧义            │
  ├─────┼──────────────────────────────────────────┼─────────┼──────────────────────────────────┤
  │ X6  │ RC/RS 字段宽度应为 1 bit                 │ info    │ 通常 RC/RS 只用于单 bit flag     │
  └─────┴──────────────────────────────────────────┴─────────┴──────────────────────────────────┘

  五、中断配置（Interrupt）

  ┌─────┬───────────────────────────────────┬─────────┬────────────────────────────┐
  │  #  │               规则                │  级别   │            示例            │
  ├─────┼───────────────────────────────────┼─────────┼────────────────────────────┤
  │ I1  │ source 字段必须有对应 enable pair │ error   │ 孤立的 source 无法屏蔽     │
  ├─────┼───────────────────────────────────┼─────────┼────────────────────────────┤
  │ I2  │ enable 字段必须有对应 source      │ warning │ 孤立的 enable 无意义       │
  ├─────┼───────────────────────────────────┼─────────┼────────────────────────────┤
  │ I3  │ source 和 enable 位宽必须一致     │ error   │ source 3-bit、enable 2-bit │
  ├─────┼───────────────────────────────────┼─────────┼────────────────────────────┤
  │ I4  │ source 字段 access 应为 W1C/RC    │ warning │ 其他类型难以做中断清除     │
  ├─────┼───────────────────────────────────┼─────────┼────────────────────────────┤
  │ I5  │ enable 字段 access 应为 RW        │ warning │ RO enable 无法动态关       │
  └─────┴───────────────────────────────────┴─────────┴────────────────────────────┘

  六、硬件接口（Hardware Interface）

  ┌─────┬────────────────────────────────────────────────┬─────────┬───────────────────────┐
  │  #  │                      规则                      │  级别   │         示例          │
  ├─────┼────────────────────────────────────────────────┼─────────┼───────────────────────┤
  │ H1  │ 标为 input 的字段 access 必须是 RO/W1C/W1S/W0C │ warning │ 语义矛盾              │
  ├─────┼────────────────────────────────────────────────┼─────────┼───────────────────────┤
  │ H2  │ 标为 output 的字段 access 必须是 RW/WO         │ warning │ RO 标 output 无意义   │
  ├─────┼────────────────────────────────────────────────┼─────────┼───────────────────────┤
  │ H3  │ 同名 HW 信号不得重复                           │ error   │ 两个 reg 生成同名端口 │
  └─────┴────────────────────────────────────────────────┴─────────┴───────────────────────┘

  七、文档与可维护性（Documentation）

  ┌─────┬──────────────────────────────────────────────────┬──────┬────────────────────┐
  │  #  │                       规则                       │ 级别 │        示例        │
  ├─────┼──────────────────────────────────────────────────┼──────┼────────────────────┤
  │ D1  │ 寄存器应有描述字段（若 Excel 有 Description 列） │ info │ 空描述提示         │
  ├─────┼──────────────────────────────────────────────────┼──────┼────────────────────┤
  │ D2  │ Side Effect 列若填了应被引用                     │ info │ 孤立的 side effect │
  └─────┴──────────────────────────────────────────────────┴──────┴────────────────────┘
